## Initializing a New Project
**Step 1.** Text goes here.
```bash
npx create-expo-app@latest
```

**Step 2.** Install the EAS CLI. 
```bash
npm install -g eas-cli
```

**Step 2.** Text goes here.
```bash
cd scarab
```

**Step 3.** Text goes here.
```bash
npx expo start
```
